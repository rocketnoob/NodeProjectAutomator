# NodeProjectAutomator 🚀

Uma ferramenta de CLI (Interface de Linha de Comando) desenvolvida em **Node.js** para agilizar o início de novos projetos web no **Linux**.

## 🛠️ O que ele faz?
Ao rodar o script, ele cria automaticamente:
- Pasta principal do projeto.
- Subpastas: `css`, `js`, `img`, `assets`.
- Arquivo `index.html` com estrutura básica vinculada ao CSS e JS.
- Arquivos `style.css` e `script.js` iniciais.
